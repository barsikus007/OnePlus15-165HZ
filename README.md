# OnePlus 15 165Hz

Enables permanent 165Hz refresh rate in every app on OnePlus 15 devices. Compatible (tested) with OnePlus 15 running OxygenOS 16

## Install

- Download zip from **[latest release](https://github.com/barsikus007/OnePlus15-165HZ/releases/latest)**
- Flash via your module manager of choice
